## 🛠️ 如何安装使用？

我已经将插件打包成了标准的 **`极速滚动截屏 (BlitzCapture)0.1.0.crx`** 文件。

**简易安装步骤：**
1. 下载文末的 `.crx` 插件文件。
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车（进入“扩展程序”页面）。
3. 勾选右上角的 **“开发者模式”**（非常重要！）。
4. 将下载好的 `.crx` 文件直接**拖拽**到这个扩展程序页面中，点击确定安装即可。
5. 固定插件图标，点击即可开始你的极速长截图之旅！
